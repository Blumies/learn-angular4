## Prerequisites 

- install git
- install nodejs > v6

## Get Started 

- Open a terminal in the container folder you would like to start the project 
- Clone the Quickstart repository 
- ```git clone https://github.com/angular/quickstart.git``` 
- ```cd quickstart```
- ```npm install```
- ```npm start```
